## 🧾 AI Expense Tracker

AI-powered expense analytics platform that extracts structured financial insights from receipts using OCR + LLM pipelines.

**Why I built it:**  
To remove the friction of manual expense tracking and make financial data searchable, structured, and intelligent.

**Highlights:**
- OCR + LLM pipeline for receipt data extraction and validation  
- Natural language → SQL query engine for financial exploration  
- Interactive dashboards for spending insights and anomaly detection  
- Asynchronous backend processing for scalable ingestion workflows  

**Tech Stack:** Remix · FastAPI · PostgreSQL · OCR · Gemini API · GitHub Actions  

🔗 Repository: https://github.com/joshbk97/AI-Expense-Tracker

## 🩺 MADI – AI Medical Diagnostic Platform

A multi-model AI system that integrates structured patient data and medical imaging to generate explainable diagnostic insights designed to support faster clinical decision-making and reduce delays in diagnosis.

**Why I built it:**  
To explore how AI systems can make complex medical predictions more interpretable and accessible.

**Highlights:**
- Multi-model orchestration (CNNs, XGBoost, structured ML models)  
- Flask-based inference pipeline for model coordination and aggregation  
- LLM-powered explanation layer for human-readable diagnostic outputs  
- Session-based prediction tracking and result aggregation  

**Tech Stack:** Python · Flask · TensorFlow · Keras · Scikit-learn · XGBoost  

🔗 Repository: https://github.com/joshbk97/MADI
